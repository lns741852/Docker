## 打包

| Docker                                                                                    | Do       | Remark                           |
| ----------------------------------------------------------------------------------------- | -------- | -------------------------------- |
| docker container commit {container_id / container_name} {映像檔名稱}                      | 打包     |
| docker image tag {image_id / image_name} {docker_hub_account}/{新映像檔名稱}:{版本標籤}。 | 加上 Tag | 標籤默認是 latest                |
|                                                                                           |          | 加上 docker_hub_account 才能上傳 |

## 上傳

1. 登入 docker hub

| Docker                                                    | Do              | Remark                      |
| --------------------------------------------------------- | --------------- | --------------------------- |
| docker login --username {account}                         | 登入 docker hub | 首次註冊要去 reset password |
| docker image push {docker_hub_account}/{image_name}:{tag} | 上傳            |                             |
| docker image pull {docker_hub_account}/{image_name}:{tag} | 下載            |                             |
