## 下載

| Docker                   | Do         |
| ------------------------ | ---------- |
| docker pull <image_name> | 下載影像檔 |

## Image

| Docker               | Do                   |
| -------------------- | -------------------- |
| docker image ls      | 列出所有影像檔       |
| docker image rm <ID> | 刪除                 |
| docker image prune   | 刪除沒有容器的映像檔 |

## Container

| Docker                                                             | Do           | Remark   |
| ------------------------------------------------------------------ | ------------ | -------- | -------------------- |
| docker container create -p <3306:3306> --name <MySQL_01> <image_id | images_name> | 創建容器 | port 前=主機 後=容器 |
| docker container ls -a                                             | 列出所有容器 |          |
| docker container rm <container_name>                               | 刪除         |          |
| docker container rm prune -f                                       | 刪除全部     |          |
| docker container start <container_name>                            | 啟動         |          |
| docker container stop <container_name>                             | 停止         |          |
| docker container ps                                                | 列出運行中   |          |
| docker container exec -it <container_name> bash                    | 進入容器     |          |
| docker container run                                               | 創建並啟動   |          |
| docker container inspect <container_name>                          | 查看詳細資料 |          | Mounts=掛載資訊      |
