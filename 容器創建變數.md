## MySQL

1. 創建容器

```
docker container run -p 3306:3306 --name <MySQL_01> <image_id |images_name>
-e MYSQL_ROOT_PASSWORD=000000
-e MYSQL_DATABASE=demo
-e MYSQL_USER=user01
-e MYSQL_PASSWORD=123456
-v "C:\db-init-sql":"/docker-entrypoint-initdb.d"
```

| 變數                                      | Do              | Remark                                                         |
| ----------------------------------------- | --------------- | -------------------------------------------------------------- |
| MYSQL_ROOT_PASSWORD                       | root 密碼       |                                                                |
| MYSQL_DATABASE                            | 創建資料庫      |                                                                |
| MYSQL_USER                                | demo 的使用     |                                                                |
| MYSQL_PASSWORD                            | demo 使用者密碼 |                                                                |
| -v {host_path}:docker-entrypoint-initdb.d | 匯入 sql 檔     | host_path 目錄下要先存入.sql                                   |
|                                           |                 | Mysql 會在初次啟動時執行 docker-entrypoint-initdb.d 裡面的.sql |

2. 備份資料

- mysqldump -localhost -u root -p demo > /var/lib/msyql/demo_backup.sql
- docker container cp /var/lib/msyql/demo_backup.sql C:\sql\demo_backup.sql
